## Step 2: Check Calendar (via Google Calendar MCP)

Use google-calendar MCP to:
- Get events for next 7 days
- Identify fixed commitments
- Calculate available time
- Note any busy days

## Step 3: Identify Key Outcomes

Based on goals and project milestones, identify 3-5 key outcomes:

### Questions to Ask:
- What project milestone is due or should be hit?
- What goal needs attention this week?
- What's been neglected that needs focus?
- What would make this week a success?

### Outcome Criteria:
- Specific and measurable
- Achievable in one week
- Connected to a goal or project
- Aligned with values

## Step 7: Offer Actions

- "Would you like me to create these time blocks on your calendar?"
- "Should I add these ONE Things to each day's journal?"
- "Want me to set up habit tracking reminders?"

## Capacity Rules

### Red Flags (Overcommitted):
- More than 6 hours scheduled on any day
- No buffer time between blocks
- Weekend fully scheduled
- Every slot filled

### Green Flags (Balanced):
- 4-6 hours deep work per day max
- 30+ min buffer between major blocks
- At least one weekend day unscheduled
- Family time explicitly protected
- Prayer times never double-booked

### Adjustments:
If overcommitted, ask:
- "Which outcome is lowest priority?"
- "What can be deferred to next week?"
- "What can be delegated?"
